NAME         top 

        _printf - produces output according to a formaat.

SYNOPSIS         top 

       
            printf FORMAT [ARGUMENT]...

DESCRIPTION         top 

            According to the FORMAT prints ARGUMENT(s):
            The format string is a character string, with zero or more directives specifying how the arguments are converted to output. Ordinary characters (not %) are printed unchanged in the output, and the % character, followed by a conversion specifier, introduces the conversion of the passed arguments.
            The arguments are used in the order given, so they must match the conversion specifier, and it is an error if there are not enough arguments.
        
            conversion specifiers
            Characters that, preceded by the character %, specifies a type of conversion to be applied in arguments.
            d, i The int argument is printed.
            c    The int argument is converted to a char and printed.
            s    The characters from the char * argument (point to an array of characters) are printed up to a terminating null byte (not included). If the pointer is NULL, prints "(null)"
            %    A '%' is printed, nothing is converted.

RETURN  VALUE        top 

       
            The function returns the number of characters printed.
            If an output error is encountered, for example if format is null, a negative value is returned.

EXAMPLES         top 

            _printf(const char*format, ...)

            To print a string
            
            _printf("%s", "Hello world");

            To print a interger

            _printf("%d", 98);
